Detailed Description

A very simple variant type for C++.

The Boost.any class is very feature complete, but also drags a large part of the Boost library along with it. There is no easy way to separate that one concept without requiring a project to depend on Boost itself.

This class is an attempt to replicate most of the functionality of Boost.any in a self- contained manner. There currently exist some limitations, such as not being able to create an instance of type::any from another type::any instance. However this should work for most cases where one would use Boost.any.
